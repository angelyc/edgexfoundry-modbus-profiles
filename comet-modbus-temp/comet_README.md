| :---           | :---:  | :---:    | :---:    | :---:  | :---: | :---:  |
| Device Address |        | 0x0006   | 6        | Int    | 2     | Read   |
| Degrees C      | C x 10 | 0x0031   | 49       | Int10  | BIN16 | Read   |
